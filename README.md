# Portfolio

Bienvenue sur le repository de mon portfolio personnel. Ce projet présente mon profil de développeur full-stack, mes projets, mon parcours et un moyen de me contacter.
---

## Description

Ce projet a été conçu comme un terrain d’expérimentation autour de Three.js, avec l’objectif de créer un portfolio original et immersif en utilisant React.

## Stack
- React (Vite)
- Tailwind CSS
- MagicUi.design
- React Three Fiber (Three.js)
- Blender
- EmailJS
- Framer Motion
- i18next
